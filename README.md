# 🧠 BrainMemory-Plugin 类脑长记忆插件
基于人脑记忆原理与认知科学打造的大模型长记忆系统，给任何大模型装上可自主控制的「长期记忆大脑」，支持**纯本地/纯云端/混合存储**一键切换，数据100%自主可控。

## ✨ 核心功能特性
- 🧠 **类脑记忆全链路**：复刻人脑「编码-巩固-存储-提取-遗忘」全流程，基于主动提取、间隔重复、精细编码等科学记忆法打造
- 🔒 **存储完全可控**：支持3种存储模式一键切换，本地模式数据100%存在你的硬盘，云端模式端到端加密，密钥仅你持有
- 🤖 **全模型兼容**：支持所有OpenAI兼容接口的大模型（GPT、通义千问、智谱AI等），以及本地开源模型（Llama3、Qwen2、通义千问开源版等）
- 🚀 **开箱即用**：提供一键启动的Python API、可视化Web界面，无需复杂配置，3分钟即可上手
- 📦 **生产级规范**：模块化设计，支持二次开发、API服务部署、多端同步
